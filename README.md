# Futuristic Game

**Futuristic Game** is an educational VR experience designed to test and reinforce core computer science skills. Set in a sleek, futuristic environment, players progress through multiple levels, each introducing new challenges and interactive learning moments.

The game supports:
- **VR mode** using Oculus/Meta Quest headsets (via Link cable or Air Link)
- **Non-VR mode** using standard mouse and keyboard controls

---

## Features
- Immersive **futuristic sci-fi** environment
- **Multiple levels** with progressive challenges
- Designed for **educational learning** through gameplay
- Playable **with VR or without VR**

## How to Install and Run

### 1. Download the Game
Download the entire game folder, which includes:
- `Futuristic Game.exe`
- `Futuristic Game_Data` folder (or similarly named Unity data folder)

### 2. Extract the Folder
If the download is a `.zip` file, extract it **before** running the game.

---

### **Running in VR Mode (Oculus/Meta Quest)**
1. Connect your headset using a **Link cable** or enable **Air Link**.
2. Start the **Oculus PC App**.
3. Put on your headset and enter **Oculus Link / Rift Mode**.
4. On your PC, open the game folder.
5. Double-click **`Futuristic Game.exe`**.
6. The game will automatically launch in VR.

---

### **Running in Non-VR Mode (Keyboard + Mouse)**
1. Open the game folder.
2. Double-click **`Futuristic Game.exe`**.
3. The game will open on your monitor in standard desktop mode.

| Pause / Menu | Esc |

---

## Troubleshooting
- **Game not launching in VR:**  
  Make sure Oculus Link is enabled **before** starting the game.
- **Controllers not responding:**  
  Check that the Oculus controller profile is active.
- **Performance issues:**  
  Close background apps or lower graphics quality in system settings.
